Univeristy Trip

The Univeristy Trip was to Blecthely Park the government institution which is know for the code breaking task for the employees there. Alan Turing a famous mathmematician who is responsable for the turing machine that managed to crack the enigma code that the Nazis used to convey messages for the radio.

Alan Turing made the turing machine the first programmable digital electronic computer which alowed the machine to be programmed to decifer the codes after the Nazis had changed the key of the code. Alan Turing's machine is also used to refer to Turing Complete which means a program or programming language that can be used to create a program that can solve any equation. On my trip we toured around the park and learnt a lot about the story of Blecthely Park and computer engineering at the time of WW2.
